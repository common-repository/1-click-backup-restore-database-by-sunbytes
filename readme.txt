== Description ==
*Caution!*
This plugin is still on developing & testing without any support. Please don't use for live site.

With 1-Click Backup & Restore Database plugin, you can export your database for backup, and restore it anytime. This plugin will not write anything to your option, and easy for one or group developers while developing website.

== Installation ==

**From your WordPress dashboard**

1.    Visit ‘Plugins > Add New’
1.    Search for ‘1-click Backup & Restore Database’
1.    Click Install Now
1.    Activate it from your Plugins page

**From WordPress.org**

1.    Download sb-one-click-update-restore (.zip file)
1.    Go to your WordPress dashboard and visit ‘Plugins > Add New’
1.    Click ‘Upload Plugin’ and upload the sb-one-click-update-restore (.zip file)
1.    Activate it from your Plugins page
